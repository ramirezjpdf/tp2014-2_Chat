NOVO Protocolo de Cliente-Servidor

Com base no modelo proposto de comunicação entre servidor-broker-cliente...

Mensagem de Chat de Sender para Receiver.
	[CHAT]Sender;Receiver;Mensagem.
	
Lista de clientes no servidor, enviada para os Clientes.
	[LISTA]Cliente1;Cliente2;Cliente3;...;ClienteN.
	
Cliente Sender pede permissão de Cliente Receiver para começar conversa com ele.
	[CHATASKPERMISSION]Sender;Receiver.
	
Cliente Receiver aceita pedido de Cliente  para começar conversa.
	[CHATGIVESPERMISSION]Receiver;Sender.
	
Cliente B recusa pedido de Cliente A para começar conversa.
	[CHATDENIESPERMISSION]Receiver;Sender.
	
Cliente A avisa para Cliente B da criação do Chat para os dois.
	[CHATCREATED]Sender;Receiver.
	
Cliente A encerra conversa com Cliente B.
	[CHATEND]Sender;Receiver.
	
Cliente A encerra conversa com todo mundo.
	[CHATEND]Sender.
